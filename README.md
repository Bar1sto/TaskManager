TaskManager — это консольное приложение для управления задачами. Оно предоставляет пользователю возможность:

Добавлять задачи.
Удалять задачи (по ID или категории).
Редактировать существующие задачи.
Просматривать список всех задач.
Фильтровать задачи по категории.
Искать задачи по ключевым словам в названии или описании.
Каждая задача имеет следующие атрибуты:

task_id: уникальный идентификатор задачи (генерируется автоматически).
title: название задачи.
description: описание задачи.
category: категория задачи.
due_date: срок выполнения задачи.
priority: приоритет задачи (низкий, средний, высокий).
status: статус задачи (по умолчанию: "Не выполнена").
Структура проекта

Классы:
Task — класс для работы с отдельной задачей.
Атрибуты: task_id, title, description, category, due_date, priority, status.

Connect — класс для работы с JSON-файлом:

Загрузка данных из файла.
Сохранение данных в файл.
Основные функции приложения

Загрузка данных из JSON
Метод load(file: str) загружает список задач из JSON-файла. Если файл пуст или отсутствует, возвращается пустой список.

Сохранение данных в JSON
Метод save(file: str, tasks: list) отвечает за сохранение списка задач в JSON-файл.

Добавление задачи
Метод add_task() позволяет пользователю добавить новую задачу, запросив данные о ней (название, описание, категорию, срок выполнения и приоритет).

Удаление задачи
Метод remove_task() предоставляет два способа удаления:

По ID задачи.
По категории (удаляются все задачи, относящиеся к указанной категории).
Редактирование задачи
Метод change_task() позволяет изменить все параметры задачи (название, описание, категорию, срок выполнения, приоритет и статус) по её ID.

Просмотр всех задач
Метод list_task() выводит список всех задач. Если задач нет, выводится соответствующее сообщение.

Фильтрация по категории
Метод list_tasks_category() выводит задачи, относящиеся к указанной категории.

Поиск по ключевым словам
Метод search_tasks() ищет задачи по ключевым словам, которые могут содержаться в названии или описании.

Этапы разработки

Создание базовых функций:

Разработка методов для работы с JSON-файлом.
Реализация добавления, удаления и редактирования задач.
Интеграция объектно-ориентированного подхода:

Введение классов Task и Connect.
Перенос функций в методы классов.
Реализация пользовательского интерфейса:

Создание меню командной строки для удобного взаимодействия пользователя с задачами.
Написание тестов:

Разработка тестов для проверки корректности работы всех методов и классов.
Требования к окружению

Python версии 3.13.
Библиотека json (встроена в стандартную библиотеку Python).
